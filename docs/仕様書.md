## 1. プロダクト概要

テクノ中部社長向けの経営参謀AIシステム。  
複数の専門エージェントが熟議を行い、経営課題に対して「第三の解」を提示する。  
FIRAの代替品として、テクノ中部の文脈に特化して構築する。

|--------|------|

---

## 3. 機能要件

### 3.1 熟議機能（コア）

- ユーザーが経営課題をテキストで入力できること
- 複数の専門エージェントが3ラウンドの熟議を行うこと
  - Round 1：各エージェントが独立して回答（相互参照なし）
  - Round 2：全意見を共有した上で反論・深掘り
  - Round 3：統合エージェントが「第三の解」を起草
- 熟議結果をユーザーに提示できること

### 3.2 記憶機能

- 過去の議論をSupabaseに永続化できること
- 次回の議論に過去の文脈を注入できること
- 会社基本情報（事業概要・課題感等）を管理できること

### 3.3 API

- POST /debate で熟議を非同期実行できること
- GET /debates で過去の議論一覧を取得できること

---

## 4. 非機能要件

| 項目 | 要件 |
|------|------|
| 応答時間 | 1回の熟議を60秒以内に完了 |
| コスト | 1回の熟議を$0.20以内に抑える |
| セキュリティ | APIキー・Supabase資格情報を環境変数で管理 |
| データ | 入力データをモデル学習に使用しない |
