# FIRA（フィーラ）事前レクチャー資料

**目的：** ハピネスプラネット社デモ来訪に先立つ基礎理解

---

## はじめに

来月、ハピネスプラネット社の社長が「FIRA（フィーラ）」という新しいAIサービスのデモに来訪されます。  
本資料は、デモ当日の議論を実りあるものにするため、FIRAの基本を事前に把握していただくための資料です。  
AIの専門知識は不要です。要点のみ、平易な言葉でまとめております。

---

## 1. いまのAI（生成AI）とは何か

### ひと言で言えば

**「膨大な過去データを検索して、それらしい答えを組み合わせる技術」** です。

### 具体例

- ChatGPTに「新製品のキャッチコピーを考えて」と入力すると、過去に世の中で使われたコピーのパターンを参考にして、それらしい文章を返してくれます。
- 「過去のデータの中に正解を探しに行く」仕組みです。

### 得意なこと・苦手なこと

| 得意 | 苦手 |
|------|------|
| 文章の要約・翻訳 | 「今まで誰も考えたことがない」アイデアを生む |
| 既知の知識の整理 | 「正解のない経営判断」に深く踏み込む |
| メール・資料の下書き | 対立する意見を超えた「第三の解」を見つける |

> **要するに：** 今のAIは「秘書」として優秀ですが、「参謀」にはなれません。

---

## 2. FIRAは何が違うのか ── 「創造AI」とは

### 開発者の言葉

FIRAを開発した日立製作所フェロー・矢野和男氏は、こう言い切っています。

> **「生成AIの時代は終わった」**

### 生成AI vs. 創造AI（FIRA）

| 比較項目 | 生成AI（ChatGPT等） | FIRA（創造AI） |
|----------|---------------------|----------------|
| 仕組み | 過去データから「それらしい答え」を検索 | 専門AIどうしが「議論」して新しい意味を生み出す |
| 得意な問い | 「○○を教えて」 | 「正解のない経営課題を深く考えたい」 |
| 成長するか | しない | する（使うたびに賢くなる） |
| たとえ | 物知りな秘書 | 各分野の専門家が集まった参謀会議 |

---

## 3. FIRAの仕組み ── 600人の「異能」が議論する

### イメージ図（例え話）

```
社長が課題を入力
        ↓
【経営戦略の専門AI】【哲学の専門AI】【歴史の専門AI】
【心理学の専門AI】 【法律の専門AI】 【文化人類学の専門AI】
        …（600種類）…
        ↓
各AIが自律的に意見を出し合い・議論する
        ↓
ファシリテーターAIが議論を整理・深化させる
        ↓
「今まで気づかなかった視点」「第三の解」が浮かび上がる
```

### 大切なポイント

1. **「異能」と呼ばれる専門AIが600種類以上**  
   経営・哲学・歴史・心理学・法律・芸術…多様な分野の知見が一堂に会します。

2. **AIたちが「自律的に」議論する**  
   人間が指示を出すのではなく、AIどうしが自らテーマに沿って話し合います。

3. **使うたびに成長する**  
   AlphaGo（囲碁AI）が対局を重ねて強くなるように、FIRAも議論を積み重ねて深化します。

## 4. 何に使うのか ── 経営者・経営企画向けの活用シーン

FIRAは主に「付加価値の高い経営判断」の場面で力を発揮します。

### 活用シーン例

| シーン | 使い方の例 |
|--------|------------|
| **経営の壁打ち** | 「次の5年をどう生き残るか？」を深く考えたい |
| **中期経営計画の策定** | 複数のシナリオを多角的に検討したい |
| **新規事業の構想** | 誰も思いつかないような事業アイデアを出したい |
| **競合・市場分析** | 複数の視点から業界の未来を描きたい |
| **リスク対応** | 最悪のシナリオを洗い出し、対策を立てたい |

### すでに使っている企業

- **日立製作所**（中期経営計画策定）
- 製造業・金融機関・建設会社・エネルギー会社・流通業など

> **日立製作所の評価：**  
> 「中期経営計画の策定プロセスにおいて、FIRAの多角的な視点が大きな価値をもたらした」

## 5. セキュリティは大丈夫か

経営上の機密情報を入力することになるため、セキュリティは重要な確認事項です。

- 通信データ・保存データはすべて**暗号化**
- FIRAに入力した情報は**AIの学習データとして使われない**（データ汚染なし）
- **PDFアップロード**対応（社内資料をそのまま読み込ませることが可能）
- アクセス権限の管理が可能
