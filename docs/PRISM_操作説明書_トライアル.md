## 1. PRISM とは

PRISM（Parallel Reasoning Intelligence System for Management）は、経営課題に対して 5 体の専門 AI エージェントが独立した視点から熟議を行い、**「第三の解」** と呼ぶ確定的提案を導出する経営参謀 AI システムです。

| エージェント | 役割 |
|---|---|
| 経営戦略家 | 中長期戦略・競合ポジション |
| CFO・財務 | 投資対効果・財務リスク |
| 技術エンジニア | 実現可能性・技術的制約 |
| 市場アナリスト | 市場動向・顧客ニーズ |
| リスク・法務 | 規制リスク・法務上の懸念 |

熟議は **Round 1（独立意見）→ Round 2（反論・深掘り）→ 第三の解** の 3 段構成で進みます。

## 3. 熟議の開始

### 入力項目

| 項目 | 入力内容 | 例 |
|---|---|---|
| セッション名 | 熟議のタイトル（管理用） | 中期事業戦略 2027 |
| 経営課題 | **会社の前提情報＋熟議させたい問い**（下記参照） | 下記テンプレート参照 |
| 熟議モード | 所要時間に応じて選択 | 下記参照 |

### ⚠️ 精度を上げる重要なコツ——前提情報を必ず添えてください

PRISM は**入力された情報のみ**を頼りに熟議します。  
会社の前提情報なしに経営課題だけを入力すると、  
業種・規模・事業環境を知らないまま回答するため、的外れな提案になることがあります。

**推奨入力テンプレート：**

```
【会社の前提】
・業種・事業内容：〇〇
・規模：売上〇〇億円、従業員〇〇名
・主力製品・サービス：〇〇
・主要顧客・販売先：〇〇
・現在の状況・強み：〇〇
・避けたい方向（あれば）：〇〇

【経営課題】
〇〇をどう判断すべきか？
```

**入力例の比較：**

| | 入力例 | 回答の傾向 |
|---|---|---|
| ❌ 前提なし | 「5 年後の主力事業をどう描くべきか」 | 業種不明のまま一般論を回答 |
| ✅ 前提あり | 「当社は東海地区の建設機械部品メーカー。売上 100 億円・従業員 300 名。主力は油圧シリンダー（国内シェア 15%）。主要顧客は国内建機メーカー 3 社。この前提で、5 年後の主力事業をどう描くべきか」 | 御社の実情に即した具体的提案 |

> **ご参考：** 2 回目以降の熟議では、過去の議論が自動的に文脈として参照されます。  
> 初回は特に丁寧な前提情報の記入をお勧めします。

### 熟議モードの選択

| モード | 所要時間 | 推奨場面 |
|---|---|---|
| **早足** | 約 30 秒 | 素早く方向感を掴みたい場合 |
| **常足** | 約 2 分 | 通常の経営判断 |
| **熟考** | 約 5 分 | 重大な投資・戦略転換の判断 |

入力後、**「熟議を開始」** ボタンを押してください。

## 4. 熟議画面の見方

熟議が始まると、5 体のエージェントがリアルタイムで発言を埋めていきます。

```
┌─────────────────────────────────────┐
│ Round 1 — 独立意見（5体同時発言）      │
│ ┌──────┐ ┌──────┐ ┌──────┐ ...     │
│ │戦略家 │ │ CFO  │ │ 技術 │         │
│ └──────┘ └──────┘ └──────┘         │
│ 多様性スコア Φ = 0.72               │
├─────────────────────────────────────┤
│ Round 2 — 反論・深掘り               │
│ （Round 1 の危険な前提への反論）       │
├─────────────────────────────────────┤
│ 第三の解                            │
│ 結論 / 論拠 / 短期アクション /        │
│ 中期ロードマップ / 少数意見           │
├─────────────────────────────────────┤
│ 失敗シナリオ（Pre-mortem）           │
└─────────────────────────────────────┘
```

---

## 5. 第三の解の読み方

| 項目 | 内容 |
|---|---|
| **結論** | PRISM が導出した最善の一手（1〜3文） |
| **論拠** | 各エージェントが結論を支持した根拠 |
| **短期アクション** | 即実行すべき施策 |
| **中期ロードマップ** | 3〜6 ヶ月の実行計画 |
| **少数意見** | 多数意見と異なる視点（反対意見の記録） |
| **創造性評価（Guilford）** | 流暢性・柔軟性・精緻性・独自性を 5 点満点で評価 |
| **失敗シナリオ** | この判断が失敗する原因上位 5 件（Pre-mortem） |

> **免責事項：** 本提案は PRISM が熟議を経て導出した最善の一手です。最終判断は社長が下されるものです。

## 6. 会社プロフィールの設定

PRISM はあなたの会社の情報を記憶し、すべての熟議に自動で反映します。  
初回ログイン時に自動的にプロフィール画面へ案内されます。

### 設定方法

ヘッダーの **「🏢 会社プロフィール」** をクリックしてください。

#### URLから自動入力（おすすめ）

| 手順 | 操作 |
|---|---|
| 1 | 会社の公式WebサイトURLを入力欄に貼り付ける |
| 2 | **「🔍 URLから自動入力」** ボタンをクリック |
| 3 | 業種・製品・顧客・強みが自動で埋まる（10〜20秒） |
| 4 | 内容を確認・補記する（売上規模・避けたい方向など） |
| 5 | **「保存する」** をクリック |

#### 手動入力

以下の項目を直接入力することもできます。

| 項目 | 入力例 |
|---|---|
| 業種・事業内容 | 東海地区の建設機械部品メーカー |
| 規模 | 売上100億円・従業員300名 |
| 主力製品・サービス | 油圧シリンダー（国内シェア15%） |
| 主要顧客・販売先 | 国内建設機械メーカー3社 |
| 現在の状況・強み | 国内シェアトップだが市場縮小傾向。設計・加工技術に強み |
| 避けたい方向 | 自動車部品参入は過去に検討済み・断念 |

> **ご参考：** プロフィールは後からいつでも更新できます。  
> 一度設定すれば、以後の熟議はすべて自動で会社情報を踏まえた回答になります。

## 8. 音声機能（任意）

### 音声入力（経営課題の入力）

マイクボタン（🎤）をクリックすると、経営課題を音声で入力できます。

> ⚠️ **注意（トライアル期間中のみ）：** 音声は Google のサーバーで認識されます。機密性の高い課題はテキスト入力をご利用ください。正式導入後はオンデバイス音声認識（外部送信なし）に換装いたします。

### 音声読み上げ（熟議結果の朗読）

熟議画面の **「🔇 音声OFF」** ボタンをクリックすると **「🔊 音声ON」** に切り替わり、エージェントの発言と第三の解の結論を順次読み上げます。
