# 用語解説：arXiv・CFIM・AQI とは

**作成日：** 2026年6月10日  
**対象：** テクノ中部社長

---

## この資料について

PRISM の開発・論文化に関する議論の中で、聞き慣れない専門用語が登場します。本資料はその用語を平易な言葉で解説するものです。

---

## 1. arXiv（アーカイブ）

### 一言で言うと
> **「世界中の研究者が論文を事前公開する無料の図書館」**

### 詳しく

arXiv（アーカイブ）は、米国コーネル大学が運営する学術論文の**プレプリントサーバー**です。

| 項目 | 内容 |
|---|---|
| 運営 | コーネル大学（米国） |
| URL | arxiv.org |
| 公開方法 | 誰でも無料で投稿・閲覧可能 |
| 分野 | 物理・数学・コンピュータ科学・AI など |

### なぜ使われるのか

通常の学術論文は、専門家による「査読」（審査）を受けてから雑誌に掲載されます。この査読には半年〜2年かかることもあります。

その間に「他の研究者が先に同じ研究を発表してしまう」というリスクがあるため、研究者は査読前に arXiv へ論文を投稿して**タイムスタンプ（発表日の証明）を確保**します。

```
研究完了
   ↓
arXiv 投稿 ← ここでタイムスタンプ確定（「○月○日に発表した」という証拠）
   ↓
査読（数ヶ月〜2年）
   ↓
学術誌掲載
```

### PRISM との関係

FIRAの競合理論「AQI」はすでに arXiv に投稿済みです（2026年2月）。PRISM の理論「CFIM」を arXiv に投稿することで、「PRISMが独立して開発した理論である」という証明を確立できます。

## 2. CFIM（シーフィム）

### 一言で言うと
> **「PRISMがなぜ『第三の解』を生み出せるのかを説明する理論」**

### 正式名称

| 略称 | 英語名 | 日本語名 |
|---|---|---|
| CFIM | Cognitive Frame Interference Model | 認知枠干渉モデル |

### 詳しく

人が問題を考えるとき、人それぞれに「認知枠（物事の見方・切り口）」があります。

- 財務担当者の認知枠：「コストと利益」
- 営業担当者の認知枠：「顧客と関係」
- 技術担当者の認知枠：「実現可能性とリスク」

普通のAI（ChatGPT等）は、一つの視点で考えます。しかし PRISMは「5つの異なる認知枠を持つエージェント（AI）が3ラウンドの討論を行う」ことで、**どの認知枠だけでも到達できなかった解（第三の解）を生み出します**。

### 波の干渉というたとえ

水面に石を2つ投げると、波紋が重なって「新しい波形」ができます。CFIMはこれと同じ原理です。異なる認知枠が「干渉（ぶつかり合う）」ことで、もとの枠にはなかった答えが浮かび上がります。

```
エージェントA（財務の枠）
           ↘
            干渉 → 「第三の解」が創発する
           ↗
エージェントB（技術の枠）
```

### PRISMの5体エージェント

| エージェント名 | 認知枠 |
|---|---|
| 戦略家 | 長期ビジョン・競争優位 |
| 財務家 | コスト・ROI・リスク |
| 現場責任者 | 実行可能性・現場負荷 |
| 外部顧問 | 第三者視点・業界標準 |
| 批評家 | 失敗要因・Pre-mortem |

### 学術論文化の意義

CFIMを学術論文として発表することで：
1. 「PRISMの熟議は理論的裏付けのある手法だ」という信頼性が増す
2. テクノ中部グループへの導入提案で「世界に発信された経営参謀AI理論」として説明できる
3. FIRAの競合理論AQIに対して優位に立てる

## 3. AQI（エーキューアイ）

### 一言で言うと
> **「FIRAが提唱する、CFIMの競合理論」**

### 正式名称

| 略称 | 英語名 | 日本語名 |
|---|---|---|
| AQI | Algebraic Quantum Intelligence | 代数量子知能 |

### 詳しく

AQIは日立製作所のグループ会社「Happiness Planet（ハピネスプラネット）」が開発した理論です。FIRAのAIエンジンの理論的根拠として、2026年2月にarXivへ論文を投稿しました。

### AQIが使う概念（簡略）

AQIは量子力学の数式（ヒルベルト空間・交換子など）を借用して、AIの創造的思考を説明しようとしています。しかし実際の実装は「知的財産の制約により公開できない」として非公開です。

### CFIMとの比較

| 比較軸 | AQI（FIRA） | CFIM（PRISM） |
|---|---|---|
| 理論の透明性 | 核心部分が非公開 | 完全に公開・再現可能 |
| 実装 | 非公開（特許化） | 標準APIで実装済み |
| 独立検証 | 第三者検証なし | 誰でも追試できる |
| 著者 | 日立グループ29名のみ | 独立した立場から発表 |
| 論文の査読 | 未査読（arXivのみ） | 未査読（arXivのみ）※同条件 |
| 実運用データ | なし（製品先行・理論後付け） | テクノ中部での実証を予定 |

> **重要：** AQIが「間違っている」というわけではありません。ただし「論文を出したが中身を見せない」という状態であり、学術的な検証ができません。CFIMはその対極に立ち、完全な透明性・再現性を強みとします。

## 4. その他の用語

### プレプリント

査読（専門家による審査）を受ける前に公開された論文の草稿。arXivに投稿されるのはプレプリントです。「プレ（pre）」は「前に」という意味。

### 査読（ピアレビュー）

論文が学術誌に掲載される前に、同分野の専門家2〜3名が内容を審査するプロセス。査読を通過した論文は信頼性が高いとされる。

### Stage 2（論文化の第2段階）

CFIMの論文化は2段階で進みます：

```
【Stage 1】（任意・保険）
  → 会社名を匿名にした理論のみのプレプリントをarXivへ投稿
  → 目的：タイムスタンプだけ確保する「IP保険」
  → 所要期間：約2週間

【Stage 2】（本命）
  → テクノ中部グループの実運用データを加えた完全版論文
  → AQIに対して「実装・実証済み」で正面対抗
  → 目標：2027年上半期・NeurIPS等の査読誌へ投稿
```

### Toubia教授（トゥービア）

Olivier Toubia（オリヴィエ・トゥービア）教授。Columbia Business School（米国・コロンビア大学経営大学院）の教授で、「AIと経営意思決定」「創造性の計算モデル」を専門とします。

CFIMの論文に外部研究者として参加いただくことで、「独立した学術機関が検証した」という信頼性が加わります。
